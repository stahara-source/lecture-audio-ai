# 講義音声ノートAI 仕様書

## 1. 画面設計（UI構成）
- **設定エリア**: APIキー入力・保存ボタン
- **入力エリア**: 講義名（テキスト入力）、音声ファイル選択（ドラッグ&ドロップ対応）
- **アクション**: 「解析を開始する」ボタン
- **表示エリア**: 3枚の結果カード（3行要約、重要ポイント、文字起こし全文）

## 2. システム挙動
- **API連携**: Gemini 2.0 Flashを使用し、音声から直接テキスト生成。
- **データ加工**: AIの回答を見出し（【要約】等）で分割し、それぞれのカードに反映。
- **永続化**: APIキーをブラウザのlocalStorageに保存。
